🟢 波比学习交流论坛 (Bobi LAN Chat) 

> 一款专为无外网/机房环境打造的轻量级、零配置、防漏读的局域网即时通讯软件。

🌟 项目简介

本项目完全基于 Python 原生生态开发，旨在解决局域网环境下同学们无法高效沟通、共享资料的问题。
客户端与服务端均支持“单文件 (.exe) 绿色分发”模式，用户拿到 U 盘拷走的软件双击即可使用，无需配置任何 Python 或第三方库环境。

✨ 核心特性

🚀 零配置 UDP 智能雷达：客户端启动后，自动在局域网内发送 `DISCOVER_BOBI` 广播寻找服务器，30秒内全自动握手连接。
💻 旗舰级可视化主控台：服务端告别命令行黑框，拥有独立可视化 GUI，支持“一键开服”与“上帝视角”实时日志监控。
💬 微信级聊天体验：
支持图文混排，气泡聊天流设计。
内置基于 Pillow 的高清无极缩放图库（支持鼠标滚轮缩放、拖拽平移）。
支持 P2P 局域网高速文件/资料直连共享。
🔒 沉浸式双击私聊**：在右侧在线名单中双击同学头像，即可唤起专属私聊窗口，数据独立路由隔离。
🔔 防漏读强提醒系统**：自建 Windows 右下角卡片式 Toast 弹窗弹射引擎，机房无声音也不会错过任何私信。
🎨 纯净 UI 与圆润视觉**：底层重写 Tkinter 渲染逻辑，实现全局平滑滚动条与自适应彩色圆角头像。

技术栈

网络通信: 原生 `socket` (TCP 长连接 + UDP 广播发现协议)
GUI 框架: `Tkinter` + 自定义事件钩子 (解决多窗口滚轮路由冲突)
图像引擎: `Pillow` (负责图片压缩、Base64 编解码与圆形蒙版遮罩裁切)
数据持久化: `SQLite3` (服务端 `bobi_forum.db` 数据持久化)
打包分发: `PyInstaller` (无控制台静默打包)

🚀 运行与使用教程

方案一：开箱即用（推荐，给同学发这个）
无需安装任何环境，完全绿色免安装！

1. 获取软件：前往本仓库右侧的 [Releases] 页面，下载最新打包好的服务端与客户端 `.exe` 文件。
2. 第一步（一键开服）：在机房找一台电脑作为主机，双击运行服务端 `.exe`。界面会亮起绿灯并显示本机的局域网 IP。
3. 第二步（同学连接）：把客户端 `.exe` 用 U 盘或机房广播发给其他同学。同学双击打开后，软件会自动通过 UDP 雷达搜寻到刚才那台主机并弹出登录框。
4. 第三步（注册开聊）：随意输入账号密码即可注册进入大厅，开始摸鱼...啊不，开始学习交流！

准备环境
请确保电脑已安装 Python 3.7 或更高版本。
